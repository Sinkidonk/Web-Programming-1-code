Now find is a command line utility that searches one or more directory trees
of a file system, it can locates files and I'll be talking about some of the used for it.

find . -name 'file*' -type f

Now this tell the find command to look in the directory it is in as show by
the . the -name 'file*' tell it to look for any file that start with the name of file, and the -type f just tell it we
are only looking for files.

Now for my favorite used of it is deleting empty folder.

find . -type d -empty -delete

Now what we are doing here we have the . that show the directory your in
-type d tell the command to only look for folders, and -empty tell it to only
look for empty folders, and then the -delete tell it to delete every file it find.
